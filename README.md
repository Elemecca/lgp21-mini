# LGP-21 Mini

The LGP-21 is an early transistorized computer manufactured by
the Librascope division of General Precision starting in 1963.
If you want to see the original computer in action, Usagi Electric
created [a series of excellent videos][usagi].

This project is a miniature replica of the LGP-21. It aims to be
as close to electrically identical to the original as possible
without relying on out-of-production components like germanium
transistors.
